# Getting started

## Installation

Please check the official laravel installation guide for server requirements before you start. [Official Documentation](https://laravel.com/docs/11.x/installation)

## Functionality Routes

You can view the following API routes:

1. Get all students: `GET` /api/students
1. Create a student: `POST` /api/students
1. Get a single student: `GET` /api/students/{id}
1. Update a student: `PUT/PATCH` /api/students/{id}
1. Delete a student: `DELETE` /api/students/{id}
